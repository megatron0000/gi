UserStatisticsFactory
service in module assessmentAnalysis

Dependencies

  • $q
  • AssessmentFactory
  • AnswerRecordFactory

Description

Expõe os resultados de um usuário em qualquer prova desejada. O formato das respostas pode ser uma lista ordenada por número da questão ou uma lista organizada por matéria.

Methods

  • getEvaluatedQuestions(assessmentId, username)

    Chama AssessmentFactory e AnswerRecordFactory para depois processar dados recebidos e retornar respostas.

    Em caso de erro no recebimento, executa 

    return $q.reject();
    Parameters
    ParamTypeDetails
    assessmentIdinteger

    id único da prova

    usernamestring

    Usuário cujo resultado se deseja

    Returns
    Array<EvaluatedUserResponse>

    $Promise cujo contexto é a lista de respostas do usuário a cada questão desta prova

  • getSortedQuestions(assessmentId, username)

    Chama getEvaluatedQuestions (método deste próprio service). Ao processar os dados, separa as questões por matéria, colocando as corretamente respondidas na frente dos Array usados, e todas em ordem crescente de questionNumber

    Em caso de erro, executa: 

    return $q.reject();
    Parameters
    ParamTypeDetails
    assessmentIdinteger

    id único da prova

    usernamestring

    Usuário cujo resultado se deseja

    Returns
    Array<SortedUserResponse>

    $Promise cujo contexto é a lista de matérias desta prova, cada uma com as questões corrigidas
