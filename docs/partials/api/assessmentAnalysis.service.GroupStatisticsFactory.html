GroupStatisticsFactory
service in module assessmentAnalysis

Dependencies

  • UserStatisticsFactory
  • $q

Description

Analisa o conjunto dos usuários ou das respostas para uma prova específica. No primeiro caso, expõe respostas corretas e incorretas. No segundo, expõe nível de dificuldade das questões.

Methods

  • getAssessmentOverallPerformance(assessmentId)

    Chama UserStatisticsFactory para recuperar respostas de usuários, e as coloca num Array separado por nomes de usuário. Em caso de falha, executa: 

    return $q.reject();
    Parameters
    ParamTypeDetails
    assessmentIdinteger

    id único da prova

    Returns
    Array<IdentifiedSortedUserResponse>

    $Promise cujo contexto é a lista com usuários e suas respostas, organizadas por matéria

  • getAssessmentQuestionPerformance(assessmentId)

    Chama this.getAssessmentOverallPerformance, processando as questões da prova para retornar uma lista com a performance de cada questão (nível de dificuldade).

    Em caso de erro, executa: 

    return $q.reject();
    Parameters
    ParamTypeDetails
    assessmentIdinteger

    id único à prova

    Returns
    Array<SortedQuestionPerformance>

    $Promise cujo contexto é a lista do desempenho da questões desta prova
