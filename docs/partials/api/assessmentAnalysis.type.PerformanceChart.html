PerformanceChart
type in module assessmentAnalysis

Methods

  • setConfiguration()

    Dependencies

    • GroupStatisticsFactory

    Usa GroupStatisticsFactory para recuperar performance de todos os usuários na prova designada por selectedAssessment e construir o Array "series" usado pelo highcharts

    Returns
    Array

    Cada elemento está no formato exigido por highcharts: membro "name" e membro "data".

Properties

    integer

    Corresponde a um "id" de prova, entre aqueles descritos por RawAssessment

  • config

    Object

    Configuração de um highchart (segue padrão definido na documentação desse API)
