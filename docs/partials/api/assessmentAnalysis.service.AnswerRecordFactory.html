AnswerRecordFactory
service in module assessmentAnalysis

Dependencies

  • $http

Description

Conecta-se ao servidor para receber ou enviar gabaritos de um aluno.

    ParamTypeDetails

    Respostas preenchidas pelo usuário (que podem conter algum erro técnico)

  • getRecord(disableCache)

    Parameters
    ParamTypeDetails
    disableCacheboolean

    true para fazer um pedido sem cache. O padrão é false

    Returns
    Array<RawUserResponse>

    $Promise cujo contexto é a lista de respostas de todos os usuários (essa lista inclui todas as provas misturadas).

    Se houver erro, alerta "Erro no recebimento de dados... recarregue a página", e retorna $q.reject()

  • getUserAnswers(username, assessmentId)

    Parameters
    ParamTypeDetails

    assessmentIdinteger

    O id único da prova desejada

    Returns
    RawUserResponse

    $Promise cujo contexto é o Object de respostas do usuário na prova pedida.

    Caso haja erro de recebimento, retorna $q.reject()
