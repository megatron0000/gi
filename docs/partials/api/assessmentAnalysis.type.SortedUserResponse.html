SortedUserResponse
type in module assessmentAnalysis

Description

Note que há redundância: a matéria à qual pertencem as questões está definida tanto em SortedUserResponse.subject quanto em cada questão pertencente a SortedUserResponse.questionList

Properties

  • subject

    string

    A matéria à qual pertencem as respostas guardadas por este Object

  • correctCount

    integer

    Contador de quantas respostas estão corretas, dentre as questões armazenadas por este Object

  • questionList

    Array<EvaluatedUserResponse>

    Questões armazenadas por este Object (são questões de uma única matéria: aquela armazenada por this.subject)
