EvaluatedAnswersController
controller in module assessmentAnalysis

Dependencies

  • $scope
  • UserStatisticsFactory
  • $stateParams

Description

Existe para apresentar a resolução da prova a um usuário (com questões corrigidas). Também organiza questões por disciplina.

Properties

  • $scope.analysisBySubject

    integer

    Faz referência a duas formas de apresentação dos acertos do usuário por disciplina dentro da página de análise individual: Análise por questão e Percentual de acerto.

    Valor 1 corresponde a Análise por questão;

    Valor 2 corresponde a Percentual de acerto

  • $scope.correctCount

    integer

    Contador da quantidade de questões acertadas pelo usuário identificado por $stateParams.username na prova especificada por $stateParams.assessmentId

  • $scope.evaluatedQuestionList

    Array<EvaluatedUserResponse>

    Lista com as questões de determinada prova, da maneira como o usuário as respondeu.

  • $scope.sortedQuestionList

    Array<SortedUserResponse>

    Informações usadas para exibir ao usuário seu resultado na prova, analisando disciplinas separadamente.
