SubmissionErrorDetail
type in module assessmentAnalysis

Description

É um detalhamento de erro, relativo a um objeto do tipo RawUserResponse.

Methods

  • errorExists()

    Returns
    boolean

    Se algum erro foi encontrado (de qualquer tipo que seja)

Properties

  • templateError

    boolean

    Se houve erro durante a montagem do objeto de respostas (por exemplo, algum campo está indefinido)

  • usernameError

    boolean

    Se o usuário não escreveu seu nome

  • answerError

    boolean

    Se o usuário deixou alguma resposta em branco
